# 📊 Personal Expenses Management System
Hi this is Nihal Shaikh and it's my first project and i have almost no knowledge of coding still i created by using Ai tools.

🌟 Project Overview
This is a comprehensive desktop application designed to help individuals track, categorize, and manage their daily financial transactions. It provides a clean interface for adding expenses, viewing history, and analyzing spending patterns.

💡 Note on Development:This project was built entirely from scratch using **Generative AI (Gemini)**. As a BSc student with no prior professional coding knowledge, I leveraged AI tools to architect the logic, debug the code, and design the system. This serves as a testament to how AI can bridge the gap between an idea and a functional product.

 🚀 Key Features
1)Transaction Logging:Easily add expenses with descriptions and amounts.
2)Categorization:Organize spending (e.g., Food, Travel, Utilities).
3)Data Persistence:Saves your data locally so it’s ready every time you open the app.
4)Summary View:Quick insights into total spending.

## 🛠️ Tech Stack
1)Language:Html,Css,Javascript 
2)Logic & Debugging:** Powered by Google Gemini (Generative AI)

📖 How I Built This (The AI Workflow)
Since I started with zero coding experience, I used a "Co-Pilot" approach:
1. Requirement Gathering:I described the features I wanted to the AI in plain English.
2. Iterative Coding:The AI generated code blocks which I tested and provided feedback on.
3. Problem Solving:When errors occurred, I used AI to explain the "Why" and "How" to fix them, effectively learning the logic behind the code as I built it.
